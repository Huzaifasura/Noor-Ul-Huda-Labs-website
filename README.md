# Noor Ul Huda Labs Website

Official website for **Noor Ul Huda Kids** - An Islamic learning app for children and families.

## 🌟 Features

- Beautiful, responsive design with Islamic-themed UI
- Custom cursor with Islamic star particles
- 3D carousel for app screenshots
- Glass morphism effects
- Multiple pages: Home, Download, Contact, Privacy Policy, Terms & Conditions

## 📱 App Available On

- [Google Play Store](https://play.google.com/store/apps/details?id=com.muslimkids.noorulhuda)
- [Amazon Appstore](https://www.amazon.com/dp/B0GQTLWXDC)
- Oppo App Market
- Vivo App Store

## 📁 Project Structure

```
├── index.html          # Main landing page
├── download.html       # Download page with store links
├── contact.html        # Contact form page
├── privacy.html        # Privacy Policy
├── terms.html          # Terms & Conditions
├── styles.css          # Custom CSS styles
├── script.js           # JavaScript (carousel, custom cursor)
├── Untitled design/    # Logo variants
└── *.png               # App screenshots and store badges
```

## 🎨 Tech Stack

- HTML5
- CSS3 (Tailwind CSS via CDN)
- Vanilla JavaScript
- Google Fonts (Plus Jakarta Sans, Manrope)
- Material Symbols
